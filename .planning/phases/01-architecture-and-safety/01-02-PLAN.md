**src/crawl.ts (new file, ~140 lines):**

Create CrawlManager class that owns crawl state and orchestration. Extract ALL crawl logic from index.ts lines 37-184.

CrawlManager constructor takes: `db: Database.Database, stmts: Statements, sources: ContentSource[]`

Methods:
- `crawlSource(source: ContentSource): Promise<number>` -- the core crawl-one-source logic. If `usesGeneration` is true, does generation swap (getCurrentGeneration, newGen, insert at newGen, finalizeGeneration). If false, inserts at currentGen (blog pattern). Sets metadata timestamps and counts. Returns page count or -1 if already crawling.
- `crawlAll(): Promise<void>` -- iterates sources in order (docs first, blog second). Awaits each sequentially. This preserves the current sequencing requirement (blog reads generation set by doc crawl).
- `checkAndCrawlAll(): void` -- for each source, checks staleness against source.staleDays. If stale or no timestamp, triggers crawlAll() in background (.catch logging). This replaces both checkAndCrawl() and checkAndCrawlBlog().
- `getState(name: string): CrawlState` -- returns state for a named source.
- `isAnyCrawling(): boolean` -- returns true if any source is in "crawling" state.
- `firstRunBuildingResponse(): { content: { type: "text"; text: string }[]; isError?: boolean } | null` -- exact same logic as current firstRunBuildingResponse() in index.ts. Checks doc source state === "crawling" AND no last_crawl_timestamp metadata.

Also define and export the two ContentSource implementations as constants:

```typescript
export const docSource: ContentSource = {
  name: "docs",
  staleDays: STALE_DAYS,
  metaTimestampKey: "last_crawl_timestamp",
  metaCountKey: "page_count",
  usesGeneration: true,
  async fetch() {
    return fetchAndParse();
  },
};

export const blogSource: ContentSource = {
  name: "blog",
  staleDays: BLOG_STALE_DAYS,
  metaTimestampKey: "last_blog_crawl_timestamp",
  metaCountKey: "blog_page_count",
  usesGeneration: false,
  async fetch(db) {
    const sitemapUrls = await fetchSitemapUrls();
    if (sitemapUrls.length === 0) return [];
    const indexedSet = new Set(getIndexedBlogUrls(db));
    const newUrls = sitemapUrls.filter((url) => !indexedSet.has(url));
    if (newUrls.length === 0) return [];
    return fetchBlogPages(newUrls);
  },
};
```

CRITICAL: The crawlSource method for blog must also handle the special case where the sitemap returns URLs but after diff there are zero new URLs -- it should still update the timestamp metadata (matching current behavior on index.ts line 94). To handle this cleanly: the blogSource.fetch() returns an empty array when no new URLs, and crawlSource checks if pages.length === 0 and still sets the timestamp.

CRITICAL: The blogSource needs to track the total count (existing + new), matching line 111 of current index.ts. The fetch method should return both the pages AND the count info. Two options: (a) make fetch return metadata alongside pages, or (b) have CrawlManager handle count differently for blog. Simpler approach: for usesGeneration=false sources, CrawlManager queries the count from DB after insert (SELECT COUNT(DISTINCT url) FROM pages WHERE source = 'blog') and uses that as the count metadata value. This is slightly different from current code (which adds indexedSet.size + pages.length) but produces the same result and is simpler.

Imports: Database from "better-sqlite3", types from "./types.js", database functions from "./database.js", parser functions from "./parser.js", blog-parser functions from "./blog-parser.js", config constants from "./config.js".

Avoid circular imports: crawl.ts depends on types.ts, database.ts, parser.ts, blog-parser.ts, config.ts. Nothing depends on crawl.ts except tools/ and index.ts.

```typescript
#!/usr/bin/env node
import { McpServer } from "@modelcontextprotocol/sdk/server/mcp.js";
import { StdioServerTransport } from "@modelcontextprotocol/sdk/server/stdio.js";
import { initDatabase, prepareStatements, cleanupOrphanedGenerations } from "./database.js";
import { CrawlManager, docSource, blogSource } from "./crawl.js";
import { registerTools } from "./tools/index.js";

const server = new McpServer({ name: "anthropic-docs", version: "2.0.0" });
const db = initDatabase();
const stmts = prepareStatements(db);

const orphans = cleanupOrphanedGenerations(db, stmts);
if (orphans > 0) console.error(`[server] Cleaned up ${orphans} orphaned rows.`);

const crawl = new CrawlManager(db, stmts, [docSource, blogSource]);
registerTools(server, stmts, crawl);

async function main() {
  crawl.checkAndCrawlAll();
  await server.connect(new StdioServerTransport());
  console.error("[server] Anthropic Docs MCP server v2 running on stdio");
}

main().catch((err) => { console.error("[server] Fatal:", err); process.exit(1); });
```
