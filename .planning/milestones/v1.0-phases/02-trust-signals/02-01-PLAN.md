    - CrawlManager.getLastError("docs") returns null when no error has occurred
    - CrawlManager.getLastError("docs") returns { message, timestamp } after a crawl failure
    - Crawl is rejected (returns 0, state=failed, error stored) when generation-based source returns < 50% of previous page count
    - Crawl proceeds normally when previousCount is 0 (first crawl)
    - Crawl proceeds normally when page count >= 50% of previous
    - Blog source is never subject to page count threshold (usesGeneration=false)
    - shutdown() calls db.close() and process.exit(0)
    - Double SIGTERM does not double-close (shuttingDown guard)
  </behavior>
  <action>
1. In src/crawl.ts CrawlManager class:
   - Add private field: `private errors: Map<string, { message: string; timestamp: string }> = new Map();`
   - Add public method: `getLastError(name: string): { message: string; timestamp: string } | null { return this.errors.get(name) || null; }`
   - In `crawlSource` catch block (line ~128-136), BEFORE the existing `this.states.set(source.name, "failed")`, add: `this.errors.set(source.name, { message: (err as Error).message, timestamp: new Date().toISOString() });`
   - In the generation branch (line ~83, after collecting pages but BEFORE `finalizeGeneration`), add threshold check:
     ```
     import { MIN_PAGE_RATIO } from "./config.js";
     // After: const newGen = currentGen + 1;
     // Before: console.error(`[server] Starting ${source.name} crawl...`);
     const previousCount = parseInt(getMetadata(this.stmts, source.metaCountKey) || "0", 10);
     if (previousCount > 0 && pages.length < previousCount * MIN_PAGE_RATIO) {
       const msg = `Crawl rejected: ${pages.length}/${previousCount} pages (below ${MIN_PAGE_RATIO * 100}% safety threshold)`;
       console.error(`[server] ${source.name}: ${msg}`);
       this.states.set(source.name, "failed");
       this.errors.set(source.name, { message: msg, timestamp: new Date().toISOString() });
       return 0;
     }
     ```
   Note: threshold only applies in the `if (source.usesGeneration)` branch, so blog is automatically excluded.

2. In src/index.ts, after the `main()` call block (after line 34), add:
   ```
   let shuttingDown = false;
   function shutdown() {
     if (shuttingDown) return;
     shuttingDown = true;
     console.error("[server] Shutting down gracefully...");
     server.close().catch(() => {});
     db.close();
     process.exit(0);
   }
   process.on("SIGTERM", shutdown);
   process.on("SIGINT", shutdown);
   ```
