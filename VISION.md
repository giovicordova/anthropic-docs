# Vision

Claude Code sits down to work and already knows every page of its own documentation. The platform guides, the coding reference, the full list of what it can do with the API — all of it is right there, searchable, browsable, always fresh. It doesn't guess. It doesn't hallucinate outdated instructions. It looks things up.

The blog posts are there too. When Anthropic publishes something new — a research finding, an engineering insight, a product announcement — it lands in the same library within the week. Claude Code reads primary sources, not secondhand summaries.

Nothing runs in the cloud. The whole library lives on the local machine, quiet and fast. It updates itself in the background: checks for new docs daily, picks up new blog posts weekly. No manual syncing, no configuration, no maintenance. It just stays current.

From the outside, there's nothing to see. No interface, no dashboard. It's invisible infrastructure — a reference shelf that Claude Code reaches for every time it needs to give an accurate answer about itself or its platform. The person working with Claude Code simply gets better answers, grounded in real documentation, without ever thinking about how.
